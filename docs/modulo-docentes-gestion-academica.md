# Módulo: Docentes y Gestión Académica

Permite a los docentes gestionar evaluaciones, notas, boletines, tareas, proyectos y aulas virtuales para primaria, secundaria y universidad.

## Funcionalidades
- Gestión de evaluaciones
- Registro de notas y boletines
- Administración de tareas y proyectos
- Aulas virtuales
