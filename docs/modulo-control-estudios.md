# Módulo: Control de Estudios

Este módulo permite la gestión académica de los estudiantes, incluyendo inscripción, historial académico, control de materias y más.

## Funcionalidades principales
- Inscripción de estudiantes
- Control y seguimiento de materias
- Gestión de calificaciones
- Emisión de constancias
